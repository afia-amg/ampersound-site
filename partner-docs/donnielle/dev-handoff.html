Developer Handoff

Indigo Reimagined Website Redesign — Ampersound Media Group, July 2026

Color Palette

  • Midnight: oklch(10% 0.04 280) / ~#121417 — Hero bg, footer, dark sections
  • Deep Indigo: oklch(22% 0.12 280) / ~#1a2340 — Service hovers, CTA, nav
  • Nectar Gold: oklch(76% 0.13 65) / ~#c9a04a — Accent, CTAs, highlights
  • Warm Cream: oklch(97% 0.006 65) / ~#f7f4f0 — Primary background
  • Violet Accent: oklch(35% 0.08 300) / ~#3d3266 — Secondary accent

Typography

  • Display: Playfair Display (Google Fonts) — 400, 600, 700, 900 + italics
  • Body: Inter (Google Fonts) — 300, 400, 500, 600
  • Hero H1: clamp(3rem, 6.5vw, 5.5rem)
  • Section H2: clamp(2rem, 3.5vw, 2.75rem)
  • Body: 1rem / 1.7 line-height

Page Structure

  1. Hero (full-viewport, photo bg + overlay, centered text)
  2. Introduction (serif statement, centered)
  3. Services (2-col grid, 6 items, hover reveals indigo bg)
  4. Featured Drinks (dark bg, 3 photo cards, monthly badges)
  5. About (centered photo 16:9, text, animated stat counters)
  6. Experience (cream bg, 2x2 feature grid)
  7. Partners (logo grid by category, 2:1 cells)
  8. CTA (indigo bg, animated glow)
  9. Contact (centered header, form)
  10. Footer (midnight bg, 3-col nav, social)

Animations

  • Scroll Reveal: opacity 0→1, translateY(40px→0), 0.9s expo out
  • Staggered: 80ms delay per child
  • Service hover: ::before translateY(101%→0), 0.6s
  • Hero parallax: content shifts up on scroll
  • Stat counters: count up on intersection
  • CTA glow: scale pulse, 10s infinite
  • Nav: backdrop blur on scroll past 100px

Technical Specs

Approach: Static HTML, all CSS/JS inline, no framework

Deploy: Netlify flat HTML, no build step

External deps: Google Fonts (Playfair Display, Inter) + Lucide Icons CDN

Responsive: Single breakpoint at 900px

Colors: OKLCH throughout (provide hex fallbacks for older browsers)

Forms: Markup only, dev wires to preferred backend

Images: WebP recommended. Hero: full-bleed bg. Drinks: 3:4. About: 16:9.

Notes

  • Photo placeholders use Unsplash stock images. Replace with actual photos.
  • Partner logos: use img with object-fit:contain, max-height 48px.
  • Featured Drinks should be CMS-updateable (JSON or simple admin).
  • OKLCH needs @supports fallback or PostCSS plugin for production.
  • Add prefers-reduced-motion media query for accessibility.
